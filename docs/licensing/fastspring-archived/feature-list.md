# Feature list (bulleted)

For the FastSpring product page features section, the landing page, and the press kit. Short, accurate, no marketing puff.

---

- **Native macOS app** — Apple Silicon, Developer ID signed, notarized
- **No sandbox** — actually reaches `~/Library/Caches`, app leftovers, Docker layers, dev artifact piles
- **Triage scan** — lightweight first pass for fast wins
- **Deep Clean** — rule-based scan with per-finding safety classification (safe / review / protected)
- **Smart Uninstaller** — scrubs leftover files when you remove an app
- **Duplicate Finder** — backed by `fclones`, accurate reclaimable-bytes math
- **Dev Artifact Purge** — surgical targeting of `node_modules`, `target/`, `.gradle/`, build outputs, Docker, DerivedData
- **File Health** — orphaned files, broken symlinks, misplaced files
- **File Organizer** — rule-based with optional local-AI proposals
- **Background Items audit** — launch agents, daemons, login items
- **Local AI explanations** — on-device 1.2B MLX model explains *why* a file was flagged
- **Optional cloud AI** — bring your own Anthropic key for harder cases
- **Protected-roots policy** — `~`, `/System`, `/Applications`, `/Library` off-limits by default
- **Audit trail** — every cleanup logged, recoverable, exportable
- **Sparkle auto-updates** — delta updates for future versions
- **3-Mac activation** — one license covers all your machines
- **14-day trial** — full features during trial; scans stay free forever
- **Open source** — AGPL-3.0, build from source for free
