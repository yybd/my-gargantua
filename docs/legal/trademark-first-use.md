in [`TRADEMARK.md`](../../TRADEMARK.md). Common-law trademark rights in the US
attach from the date the mark is first used to identify goods in commerce, so
the earliest verifiable public uses are recorded here.

- **Goods:** native macOS disk-cleaning / system-maintenance / storage-
  reclamation software for developers and power users

## Clearance status

A non-certified web clearance sweep was performed in **June 2026**. Summary:

- No live US federal trademark for GARGANTUA in software classes 9 or 42 was
  found. The only federal GARGANTUA filing identified was Slamtec (Class 7,
  industrial machinery), which is unrelated goods.
- No senior commercial software product named "Gargantua" in the cleaner /
  utility / storage / backup / dev-tool space was found.
- No Warner Bros. / Paramount / Legendary "Gargantua" trademark was found; the
  Interstellar use is of a pre-existing common word and does not bar software
  use. Avoid the film's visual identity (black-hole render, poster, typography)
  in marketing.
- The mark is arbitrary-to-suggestive as applied to a cleaner and appears
  inherently distinctive / registrable.

**Caveat:** the authoritative USPTO interfaces (TESS / TSDR) blocked automated
access during the sweep, so the above is a strong signal but **not a certified
search**. Before relying on the mark commercially or filing a registration,
obtain a hand TESS/TSDR search or a flat-fee attorney clearance opinion.
